# Retail-Scales-Project📊 Retail Sales Intelligence & Forecasting System

🔥 Project Overview

This project is an end-to-end data analytics solution that analyzes retail sales data and generates actionable business insights along with forecasting.

🛠️ Tools & Technologies Used

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQLite (Database)
- Power BI (Dashboard Visualization)
- Jupyter Notebook

📈 Key Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Sales Trend Analysis
- Business Insights Generation
- Interactive Power BI Dashboard

📊 Dashboard Preview

"Dashboard" (dashboard.png)

📂 Project Structure

- analysis.ipynb → Python Analysis Code
- retail_sales_powerbi.csv → Dataset
- dashboard.png → Dashboard Output

🚀 Outcome

This project demonstrates end-to-end data analytics workflow from raw data to business insights and visualization.
